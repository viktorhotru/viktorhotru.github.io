Игровой баннер Lay's Football

Задача

Проведение охватной кампании “Футбольные фанаты” в период ЧМ 2018.

Идея

Клиент поставил задачу подготовить креатив, активно вовлекающий пользователей во взаимодействие. Мы предложили несколько вариантов с футбольной тематикой, в том числе, мини-игру.

Правила очень просты: за 15 секунд пользователь должен пробить серию пенальти. Механика игры работала по принципу одного действия, от пользователя требовалось только бить по мячу. При этом, мы добавили 2 усложняющих условия: плавающая сила удара и постоянно двигающийся вратарь.

На экране пользователь видел шкалу с силой удара, которая то возрастала, то убывала. Нужно было дождаться максимальной отметки и тапнуть по мячу для победного гола. Вратарь без остановки защищал ворота, поэтому не каждому пользователю удавалось с первого раза забить гол.

Для поддержки пользователя, периодически на экране появляются футбольные фанаты со своими фирменными кричалками.

Результаты

По результатам проведенной кампании, 76% пользователей доиграли до конца - открыли вижуал. Конверсия переходов из игры на посадочную страницу выше средней в 2 раза.
